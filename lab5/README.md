# Lab 5: Docker Basics

Репозиторий для 5-й лабораторной работы по курсу DevOps. 
В рамках задания создан Docker-образ с Nginx, который отдает статическую HTML-страницу.

## Выполненные требования

### Базовое задание:
- [x] Создан Docker-образ на базе `nginx:alpine`.
- [x] Внутри работает Nginx, отдающий статическую HTML-страницу.
- [x] Порт `54321` хоста проброшен на порт `443` контейнера (для HTTPS) и `8080` на `80` (для HTTP).
- [x] Команда запуска оформлена в виде shell-скрипта (`run.sh`).

### Дополнительное задание:
- [x] Сгенерирован самоподписанный SSL-сертификат (`generate_cert.sh`).
- [x] Nginx настроен на работу по протоколу HTTPS.
- [x] Сертификат пробрасывается в контейнер через Volume Mapping (`-v`).
- [x] Написан скрипт обновления сертификата (`update_cert.sh`), который перегенерирует cert и делает `nginx -s reload` внутри работающего контейнера.

## Как использовать

1. Склонируйте репозиторий и перейдите в папку:
   ```bash
   git clone <ссылка_на_репо>
   cd lab5
   ```

2. Сделайте скрипты исполняемыми:
   ```bash
   chmod +x *.sh
   ```

3. Запустите контейнер:
   ```bash
   ./run.sh
   ```

4. Проверьте работу:
   - HTTP: `curl http://localhost:8080`
   - HTTPS: `curl -k https://localhost:54321` (флаг `-k` игнорирует проверку самоподписанного сертификата)

5. Для обновления сертификата без остановки контейнера:
   ```bash
   ./update_cert.sh
   ```
